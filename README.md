# TicTacToe iOS

It is a classic TicTacToe game

Built using XCode 13.1 (Swift 5.5, for iOS 13.0 or later).

### How to run:

1. Clone this repo
1. Open `tictactoe.xcodeproj`
1. Build and run the project on selected device (or simulator)
